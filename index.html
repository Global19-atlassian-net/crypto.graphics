PRF Security Game

b = 0 b = 1 Adversary
k = rand(Keys) R = rand(Functions)
m ← m ∈ Inputs
c = F(k, m) c = R(m) → c
bguess ∈ {0, 1}
bguess
↓

Advantage

  • W0 = Pr[bguess = 1 | b = 0]
  • W1 = Pr[bguess = 1 | b = 1]
  • Advantage[Adversary, F] = |W0 - W1|
