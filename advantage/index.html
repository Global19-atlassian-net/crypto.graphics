Advantage

A scheme/system S is considered secure if the advantage of any efficient (polynomial-time) adversary against S in the relevant security game is negligible.

  • W0 = Pr[bguess = 1 | b = 0]
  • W1 = Pr[bguess = 1 | b = 1]
  • Advantage[Adversary, S] = |W0 - W1|
